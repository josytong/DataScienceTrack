# London Housing Case Study

## Overview
The objective of this case study is to explore and analyze London housing price data using Python and data analysis techniques. The project focuses on identifying trends in housing prices across London boroughs and understanding how property values have changed over time.

## Objectives
- Analyze housing price changes across London boroughs over time.
- Explore trends and patterns within the housing dataset.
- Compare average price increases between boroughs.
- Use data visualizations to highlight key findings.
- Identify the borough with the greatest average increase in housing prices.

## Tools and Libraries
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib

## Project Files
- `London_Housing_Case_Study.ipynb` – Main analysis notebook
- `README.md` – Project overview
- `data/` – UK_House_price_index.xlsx

## Key Findings
- Housing prices vary significantly across London boroughs.
- Some boroughs experienced faster growth than others over the study period.
- Data visualization helped identify trends and compare borough-level performance.
